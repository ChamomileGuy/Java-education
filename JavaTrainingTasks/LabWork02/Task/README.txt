RU

Описание классов и методов из пакета buildings находится в файле "02. Базовые конструкции.doc" в родительском каталоге "LabWork02".

Файл Main.java используется для создания экземпляров классов и демонстрации работы методов.

Отметим, что по заданию нельзя использовать классы из пакета java.util. 
Но для более удобного и понятного вывода объектов квартир в классе Flat был перегружен метод toString() из класса java.util.Arrays.
Иного использования пакета java.util в программе нет.
Если вы будете использовать классы пакета buildings в своей программе, то импортируйте класс java.util.Arrays, или удалите перегрузку метода toString() в классе Flat



EN

A description of the classes and methods from the buildings package is located in the file "02. Basic structures.doc" in the parent directory "LabWork02".

The Main.java file is used to create instances of classes and demonstrate how methods work.

Note that in the task it is forbidden to use classes from the java.util package.
But for a more convenient and understandable display of flat instances in the Flat class, the toString() method from the java.util.Arrays class was overrided.
There is no other use of the java.util package in the program.
If you will use the buildings package classes in your program, then import the java.util.Arrays class, or remove the toString() method overrided in the Flat class
